nanoGALLERY

jQuery plugin

 

 

Demo: multi-level navigation using HREF method.

 

 

 

 

 

Requirements:

- assign an ID to each item,

- assign the kind to each album,

- assign the ID of to album for each image.

 

Thailand Vietnam India Thailand1 Thailand2 Thailand3 Thailand4

Corresponding HTML:


<div id="nanoGallery2">
  <a href="" data-ngthumb="image_01ts.jpg" data-ngID="1" data-ngAlbumID="0" data-ngkind="album">Thailand</a>
  <a href="image_02.jpg" data-ngthumb="image_02ts.jpg" data-ngID="2" >Vietnam</a>
  <a href="image_03.jpg" data-ngthumb="image_03ts.jpg" data-ngID="3" >India</a>
  <a href="image_01.jpg" data-ngthumb="image_01ts.jpg" data-ngID="4" data-ngAlbumID="1" >Thailand1</a>
  <a href="image_01.jpg" data-ngthumb="image_01ts.jpg" data-ngID="5" data-ngAlbumID="1" >Thailand2</a>
  <a href="image_01.jpg" data-ngthumb="image_01ts.jpg" data-ngID="6" data-ngAlbumID="1" >Thailand3</a>
  <a href="image_01.jpg" data-ngthumb="image_01ts.jpg" data-ngID="7" data-ngAlbumID="1" >Thailand4</a>
</div>

Corresponding Javascript:


jQuery(document).ready(function () {
  jQuery("#nanoGallery2").nanoGallery({
    thumbnailWidth:120,thumbnailHeight:120,
    itemsBaseURL:'demonstration',
    thumbnailHoverEffect:'borderLighter',
    viewerDisplayLogo:true
  });
});
