nanoGALLERY

jQuery plugin

 

 

Demo: deep linking and back/forward navigation.

 

Set the parameter locationHash to true to activate deep linking and back/forward navigation.

Now every click on an item will change the URL. You can share these URLs with Twitter for example.

 

 

 

 

Example: deep linking and back/forward navigation.

 

Corresponding HTML:


<div id="nanoGalleryExample"></div>

Corresponding Javascript:


jQuery("#nanoGalleryExample").nanoGallery({
  userID:'cbrisbois@gmail.com',
  kind:'picasa',
  blackList:'Scrapbook|forhomepage|profil',
  thumbnailWidth: 100,
  thumbnailHeight: 100,
  thumbnailLabel: {
    display: true,
    position: 'overImageOnBottom'
  },
  locationHash:true,
  viewerDisplayLogo:true,
  thumbnailLazyLoad:true,
  thumbnailHoverEffect:[{'name':'slideUp','duration':300},{'name':'borderLighter'}]
});

  <a href="#nanogallery/nanoGalleryExample/5911347863561293937">click here</a>
  <a href="#nanogallery/nanoGalleryExample/5870136791457517345/5871255268929586546">click here</a>
  <a href="#nanogallery/nanoGalleryExample/0">click here</a>



Note: this feature can interfere with pages where hash location is already used. In particulary, if you have several nanoGALLERY on one page, this option should only be activated for one instance.
