nanoGALLERY

jQuery plugin

 

 

Demo: use image lazy load to handle a large amount of albums and images.

 

This demo shows you how to improve perfomances to manage many data.

A large Flickr account is used: Thomashawk, with about 1900 photosets.

The display time of the first page is dependant of your internet connection speed.

 

 

 

 

Example: image lazy loading.

 

Corresponding HTML:


<div id="nanoGallery"></div>

Corresponding Javascript:


jQuery("#nanoGallery").nanoGallery({
  kind: 'flickr',
  userID: '51035555243@N01',
  thumbnailWidth: 100,
  thumbnailHeight: 100,
  thumbnailLabel: {
    display: true,
    position: 'overImageOnBottom'
  },
  maxItemsPerLine:7,
  thumbnailDisplayInterval:0,
  thumbnailLazyLoad:true,
  thumbnailDisplayTransition:false,
  thumbnailHoverEffect:[{'name':'slideUp','duration':300},{'name':'borderLighter'}]
});
